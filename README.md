# Earnext AI Lead & Client Intelligence System

Versione corretta per GitHub Pages.

Correzioni incluse:
- la voce Home apre una pagina reale, non una pagina bianca;
- la lista lead a sinistra mostra i 3 lead demo anche prima del caricamento CSV;
- caricando un CSV reale, la lista viene sostituita dai dati reali;
- la funzione Report resta attiva.

## Pubblicazione
Carica `index.html`, `.nojekyll` e `README.md` nella root del repository GitHub e attiva GitHub Pages da Settings → Pages.
